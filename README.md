# Portfolio Generator

Use this Portfolio Generator to display your projects! 

## Getting Started

Clone or fork this repository

```bash
git clone git@github.com:kotalilyy/portfolio-generator.git
```

## Installing

```bash
npm install
```

## Technologies Used

- HTML

- CSS 

- JavaScript/ ES6

- Node.js

- Inquirer
