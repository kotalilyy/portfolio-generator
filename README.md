# Portfolio Generator

## Technologies Used

- HTML

- CSS 

- JavaScript/ ES6

- Node.js

- Inquirer

## Getting started

Clone this repository

```bash
git clone git@github.com:kotalilyy/portfolio-generator.git
```

## Installation

```bash
npm install
```
